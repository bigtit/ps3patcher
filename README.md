# ps3patcher
a homely port of ps3 nor/nand patcher program of  Swizzy/PS3DumpChecker from windows to gnu linux using c++

### how to use
1. git clone git@github.com:bigtit/ps3patcher.git
2. cd ps3patcher
3. make
4. ./tmain -t -f -s ps3.dump

### test environment
archlinux 4.0.5 x86_64
gcc version 5.1.0
